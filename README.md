# Student Performance Dashboard  

#Internship Project – CodeAlpha  

This project analyzes student academic performance data and presents insights using an interactive dashboard.

---

#Objective  

To identify key factors influencing student performance and present insights using data visualization techniques.

---

# Dataset Overview  

- 1000 student records  
- Features include:  
  - Gender  
  - Test Preparation  
  - Lunch Type  
  - Parental Education Level  
  - Math Score  
  - Reading Score  
  - Writing Score  

---

# Dashboard Features  

- ✔ Overall Average Score  
- ✔ Subject-wise Comparison  
- ✔ Test Preparation Impact  
- ✔ Gender-based Analysis  
- ✔ Lunch Category Comparison  
- ✔ Parental Education Analysis  
- ✔ Score Distribution  

---

#Key Insights  

- Students completing test preparation performed better  
- Reading scores slightly higher than Math  
- Higher parental education correlates with better performance  
- Standard lunch students showed better averages  

---

# Challenges & Solutions  

**Data Cleaning:** Standardized dataset and validated categories  
**KPI Selection:** Focused on performance indicators  
**Dashboard Layout:** Structured for clarity and storytelling  

---

#Tools Used  

- Power BI  
- Data Cleaning Techniques  
- Data Visualization Principles
